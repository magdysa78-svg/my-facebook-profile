# 📱 Social Profile Page - Facebook Style

مشروع بسيط لصفحة تعريفية (Profile Card) تشبه أسلوب فيسبوك، تم بناؤه باستخدام تقنيات الواجهة الأمامية الأساسية. هذا المشروع هو جزء من رحلتي في تعلم الـ **Frontend**.

## 🛠 التقنيات المستخدمة (Tech Stack)
*   **HTML5:** لبناء هيكل الصفحة.
*   **CSS3:** للتنسيق والتصميم (Flexbox, Positioning).

## 🚀 المميزات
*   تصميم عصري ونظيف.
*   صورة غلاف (Cover Photo) وصورة شخصية دائرية.
*   تنسيق مرتب للإحصائيات (أصدقاء، منشورات).
*   زر تفاعلي (Follow Button).
